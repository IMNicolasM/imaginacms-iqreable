# imaginacms-iqreable

## Install
```bash
composer require imagina/iqreable=v10.x-dev
```

```bash
composer require simplesoftwareio/simple-qrcode
```

## Enable the module
```bash
php artisan module:enable iqreable
```

## Migrations
```bash
php artisan module:migrate iqreable
```

## Seeder
```bash
php artisan module:seed iqreable
```
